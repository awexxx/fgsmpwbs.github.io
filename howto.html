Thanks for joining the Forest Grove SMP!

This read-me will get you started with installing the modpack to your mods folder.

1. Install JAVA

Minecraft has abandoned requiring Java since 1.13 and after (so close), but since we're on 1.12.2, we need it.
Navigate to the Java website and install it from there.


2. Install Minecraft Forge

You're going to need Forge to install mods.
Navigate to the Forge website and select "Installer" under Recommended.
WARNING! Click SKIP in the top right after the countdown. Do not click anything else!

3. Start Minecraft 1.12.2 & Fully install Forge

If you have not started Minecraft 1.12.2 previously (Vanilla), then you need to do so.
Head to the Minecraft Launcher, click Installations, New, then find 1.12.2 and save. Click play, and the game should launch.
Once you're at the main menu, you can just close out as you've created what Forge needs. Run the JAR file you downloaded from step 2 and install Client.

4. Install Mods

On Windows, if you press the Windows key + R, a box called "Run" should come up.
In this box, type "%appdata%" (without quotes) and you should be brought to the Roaming folder.
Go into .minecraft/mods, then drag all of the mods from forest-grove-latest.zip/mods to .minecraft/mods.

5. Play!

You have successfully installed needed mods to play!

Known Issues

Crashes upon Useful Backpacks Init

Players may experience crashes upon booting the game due to Error Code -1. We found that this is a dependency error for Useful Backpacks.
Contact Awex through a ticket and send your latest Client report from .minecraft/crash-reports.

Pixelmon "Version NULL" Issue

This error happens because you installed Pixelmon incorrectly. If you have JAR components (ex. "assets") in your mods folder, you extracted the Pixelmon mod.
Go back into the archive and only drag the Pixelmon Jar out, do not extract it as it will not load.

You can find all issues on our issue tracker.
